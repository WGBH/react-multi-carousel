# Theseusx Carousel

Fork of https://github.com/YIZHUANG/react-multi-carousel.
For original readme, [click here](./original_README.md)

## How to use

- Create a branch.
- Make any changes you need.
- Run `npm run build` to generate static files.
- Copy the entire `lib/` folder (not just its contents) over to `wgbhx/packages/carousel` and overwrite everything.
- Don't forget to create a pull request for your changes afterwards!
